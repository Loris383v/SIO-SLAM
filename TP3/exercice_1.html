Texte structuré

<p> - Paragraphe :

Ceci est un paragraphe de texte.

<blockquote> - Citation :

Ceci est une citation

<pre> - Texte préformaté :

     Texte préformaté

Listes

<ul> Liste non ordonnée :

  • Élément 1
  • Élément 2

<ol> Liste ordonnée :

  1. Premier
  2. Deuxième

Texte technique

  • <code> - Code informatique : print("Hello")
  • <kbd> - Entrée utilisateur : Ctrl + C
  • <sup> - Exposant : x2
  • <sub> - Indice : H2O
  • <abbr> - Abréviation : HTML
