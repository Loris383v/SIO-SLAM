Mémento des balises HTML basiques

Balises HTML pour la structure de base d’un document

  • <!DOCTYPE> - Détermine le type de document.
  • <html> - Définit un document HTML.
  • <head> - Contient les métadonnées du document.
  • <title> - Définit le titre du document.
  • <body> - Contient le contenu affiché dans le navigateur.
  • <nav> - Zone de navigation du site Web.
  • <section> - Regroupe des éléments.
  • <article> - Partie contenu de la page.
  • <header> - Définit l’en-tête d’une page.
  • <footer> - Définit le pied de page.

Balises de titres

<h1> à <h6> définissent des titres de niveaux différents :

  • Titre principal (h1)

  • Sous-titre (h2)

  • Sous-sous-titre... (h3)
