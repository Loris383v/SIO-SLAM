Mémento HTML

Structure de base :

<!DOCTYPE html>
<html lang="fr">
<head>
<title>Titre ici</title>
</head>
<body>
<!-- Contenu ici -->
</body>
</html>

Les Titres :

De h1 à h6, du plus important au moins important

<h1>Titre Principal</h1>
<h2>Titre Secondaire</h2>

Titre Principal


Titre Secondaire

Paragraphes :

Les paragraphes sont utilisés pour séparer les informations

<p>Ceci est un paragraphe.</p>

Ceci est un paragraphe.

Commentaires :

Les commentaires sont utilisés pour expliquer le code, ne sont pas visibles sur la page

<!-- Ceci est un commentaire -->
On ne voit rien ici, mais on verra le commentaire si on inspecte le code de la page

Liens :

Permet de rediriger vers une autre page, un autre site

<a href="https://www.exemple.com">Texte du lien</a>
Texte du lien

Divisions :

Utilisées pour grouper des éléments

<div>Contenu de la division</div>
Contenu de la division

Listes :

Permet de lister des éléments

Liste non ordonnée :

<ul>
  <li>Élément 1</li>
  <li>Élément 2</li>
</ul>
  • Élément 1
  • Élément 2

Liste ordonnée :

<ol>
  <li>Élément 1</li>
  <li>Élément 2</li>
</ol>
  1. Élément 1
  2. Élément 2

Tableaux :

Permet de présenter des données de manière structurée

<table>
  <tr><th>Colonne 1</th><th>Colonne 2</th></tr> -- Ligne 1
  <tr><td>Donnée 1</td><td>Donnée 2</td></tr> -- Ligne 2
</table>
Colonne 1 Colonne 2
Donnée 1 Donnée 2

Formulaires :

Permet de collecter des informations de l'utilisateur

<form>
  <label for="username">Nom d'utilisateur:</label>
  <input type="text" id="username" name="username">
  <input type="submit" value="Envoyer">
</form>

Sémantique :

Permet de donner un sens aux blocs dans le code

<header>En-Tête</header>
<nav>Navigation</nav>
<section>Section</section>
<article>Article</article>
<aside>Aparté</aside>
<footer>Pied de page</footer>

Éléments multimédias :

Pour intégrer des vidéos , des images, des sons

Vidéo :

<video controls><source src="video.mp4" type="video/mp4"></video>

Image :

<img src="image.jpg" alt="Description de l'image en cas de non chargement et pour les lecteurs">

Audio :

<audio controls><source src="audio.mp3" type="audio/mpeg"></audio>

Caractères spéciaux :

Pour indiquer des éléments qui pour certains seraient mal interprétés par HTML si écrits en clair

  • &copy; → ©
  • &amp; → &
  • &lt; → <
  • &gt; → >
  • &quot; → "
  • &nbsp; → espace insécable
